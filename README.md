# 🎬 YouTube Telegram Bot

Потужний Telegram бот для завантаження відео та аудіо з YouTube з мікросервісною модульною архітектурою та веб-панеллю розробника.

## 🚀 Швидкий старт

### 1. Встановлення залежностей

```bash
# Встановлення системних пакетів
sudo apt update && sudo apt install -y python3.13-venv python3-pip ffmpeg

# Створення віртуального середовища
python3 -m venv venv
source venv/bin/activate

# Встановлення Python залежностей
pip install -r requirements.txt
```

### 2. Налаштування

1. **Створіть Telegram бота:**
   - Відправте `/newbot` боту @BotFather в Telegram
   - Дотримуйтесь інструкцій для створення нового бота
   - Збережіть отриманий токен

2. **Налаштуйте конфігурацію:**
   ```bash
   cp .env.example .env
   ```
   
   Відредагуйте `.env` файл:
   ```env
   BOT_TOKEN=ваш_telegram_bot_token
   DATABASE_URL=sqlite+aiosqlite:///./youtube_bot.db
   DOWNLOAD_PATH=./downloads
   MAX_DURATION_MINUTES=30
   MAX_FILE_SIZE_MB=500
   ```

### 3. Тестування

```bash
# Запуск тестів
python test_bot.py
```

### 4. Запуск бота

```bash
python main.py
```

## 📋 Залежності

### Python пакети:
- `aiogram` - Telegram Bot API
- `yt-dlp` - YouTube завантажувач
- `sqlalchemy` + `aiosqlite` - Асинхронна база даних
- `pydantic` + `pydantic-settings` - Валідація та конфігурація
- `loguru` - Логування
- `aiohttp` + `aiofiles` - Асинхронні HTTP та файлові операції

### Системні залежності:
- `ffmpeg` - Обробка відео/аудіо
- `python3.13-venv` - Віртуальні середовища Python

## 🎮 Використання

### Основні функції:

1. **Надіслання URL** - просто надішліть посилання на YouTube відео
2. **Вибір типу** - оберіть завантаження відео або аудіо
3. **Налаштування якості** - виберіть бажану якість
4. **Відстеження прогресу** - спостерігайте за процесом завантаження
5. **Отримання файлу** - завантажте готовий файл

### 🔗 Підтримувані формати URL:
- **Стандартні:** `https://www.youtube.com/watch?v=VIDEO_ID`
- **Короткі:** `https://youtu.be/VIDEO_ID`
- **YouTube Shorts:** `https://www.youtube.com/shorts/VIDEO_ID`
- **YouTube Music:** `https://music.youtube.com/watch?v=VIDEO_ID`
- **Embed:** `https://www.youtube.com/embed/VIDEO_ID`
- **Мобільні:** `https://m.youtube.com/watch?v=VIDEO_ID`
- **З параметрами:** `https://www.youtube.com/watch?v=VIDEO_ID&t=30s`

## ⚙️ Конфігурація

### Основні налаштування:

| Параметр | Опис | За замовчуванням |
|----------|------|------------------|
| `BOT_TOKEN` | Токен Telegram бота | *обов'язковий* |
| `DATABASE_URL` | URL бази даних | `sqlite+aiosqlite:///./youtube_bot.db` |
| `DOWNLOAD_PATH` | Шлях для завантажень | `./downloads` |
| `MAX_DURATION_MINUTES` | Максимальна тривалість відео | `30` |
| `MAX_FILE_SIZE_MB` | Максимальний розмір файлу | `500` |
| `MAX_DOWNLOADS_PER_USER_HOUR` | Ліміт завантажень на годину | `10` |

### Логування:

| Параметр | Опис | За замовчуванням |
|----------|------|------------------|
| `LOG_LEVEL` | Рівень логування | `INFO` |
| `LOG_FILE` | Файл для логів | `./logs/bot.log` |

## 🏛️ Структура бази даних

### Таблиця `users`:
- `id` - Унікальний ідентифікатор
- `telegram_id` - ID користувача в Telegram
- `username` - Ім'я користувача
- `first_name`, `last_name` - Ім'я та прізвище
- `downloads_count` - Кількість завантажень
- `is_banned` - Статус блокування
- `created_at`, `last_seen` - Дати створення та останньої активності

### Таблиця `downloads`:
- `id` - Унікальний ідентифікатор
- `user_id` - Посилання на користувача
- `video_url` - URL відео
- `video_title` - Назва відео
- `download_type` - Тип завантаження (video/audio)
- `quality` - Якість файлу
- `status` - Статус завантаження
- `file_path` - Шлях до файлу
- `created_at`, `completed_at` - Дати створення та завершення

## 🐛 Відлагодження

### Часті проблеми:

1. **"Token is invalid"** - перевірте токен бота в `.env`
2. **"yt-dlp помилка"** - YouTube може вимагати автентифікацію
3. **"Database error"** - перевірте права доступу до файлу БД
4. **"FFmpeg not found"** - встановіть `ffmpeg`

### Логи:

Перевірте файл `./logs/bot.log` для детальної інформації про помилки.
