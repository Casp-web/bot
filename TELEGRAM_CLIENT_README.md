# Telegram Client на tkinter

Простий Telegram клієнт, побудований на Python з використанням tkinter та Telethon.

## 🌟 Особливості

- 📱 Підключення до Telegram через API
- 💬 Перегляд та відправка повідомлень
- 📎 Відправка файлів та медіа
- 🔍 Пошук по чатах
- ⚙️ Збереження налаштувань
- 🎨 Кольорове виділення повідомлень
- ℹ️ Інформація про чати
- 🖥️ Зручний графічний інтерфейс

## 🚀 Швидкий запуск

### Варіант 1: Автоматичний запуск
```bash
python3 run_telegram_client.py
```

### Варіант 2: Ручне встановлення
```bash
# Встановлення залежностей
pip install telethon --break-system-packages

# Запуск клієнта
python3 telegram_client.py
```

## 📋 Встановлення

### 1. Системні вимоги
- Python 3.7+
- tkinter (зазвичай входить в стандартну поставку Python)
- Активний Telegram акаунт

### 2. Встановлення системних пакетів (Ubuntu/Debian)
```bash
sudo apt update
sudo apt install python3-tk python3-pip python3-venv
```

### 3. Встановлення Python залежностей

#### Варіант A: Віртуальне середовище (рекомендовано)
```bash
python3 -m venv telegram_client_env
source telegram_client_env/bin/activate
pip install telethon
python telegram_client.py
```

#### Варіант B: Системна установка
```bash
pip install telethon --break-system-packages
python3 telegram_client.py
```

### 4. Отримання API credentials

1. Перейдіть на https://my.telegram.org/
2. Увійдіть з вашим номером телефону
3. Перейдіть в 'API Development tools'
4. Створіть нову програму:
   - **App title**: Назва вашої програми (наприклад, "My Telegram Client")
   - **Short name**: Коротка назва (наприклад, "my_client")
   - **Platform**: Desktop
   - **Description**: Опис програми
5. Скопіюйте ваш **API ID** та **API Hash**

## 🎯 Використання

### Перший запуск

1. Запустіть програму:
   ```bash
   python3 run_telegram_client.py
   ```

2. Введіть ваші дані:
   - **API ID**: Ваш API ID з my.telegram.org
   - **API Hash**: Ваш API Hash з my.telegram.org
   - **Phone**: Ваш номер телефону у міжнародному форматі (+380XXXXXXXX)

3. Натисніть **"Connect"**

4. При першому підключенні:
   - Telegram надішле код підтвердження на ваш телефон
   - Введіть код у діалоговому вікні
   - Можливо знадобиться ввести пароль двофакторної автентифікації

5. Після успішного підключення ви побачите список ваших чатів

### Основні функції

#### 📱 Підключення
- Введіть API credentials та номер телефону
- Натисніть "Connect" для підключення
- Статус підключення відображається у правому верхньому куті

#### 💬 Робота з чатами
- **Перегляд чатів**: Список всіх чатів у лівій панелі
- **Вибір чату**: Клікніть на чат для відображення повідомлень
- **Пошук чатів**: Введіть текст у поле "Search" для фільтрації
- **Інформація про чат**: Подвійний клік або кнопка "Info"

#### 📝 Повідомлення
- **Перегляд**: Історія повідомлень відображається в центральній панелі
- **Відправка**: Введіть текст у нижнє поле та натисніть "Send"
- **Гарячі клавіші**: Ctrl+Enter для відправки
- **Кольорове виділення**: Сині - ваші повідомлення, чорні - отримані

#### 📎 Файли
- **Відправка файлів**: Натисніть кнопку 📎 та виберіть файл
- **Підтримувані типи**: Зображення, документи, архіви та інші
- **Відображення**: Файли показуються з іконками та назвами

### Меню програми

#### 📁 File
- **Save Credentials**: Збереження API даних для наступних запусків
- **Clear Credentials**: Очищення збережених даних
- **Exit**: Вихід з програми

#### 👁️ View
- **Refresh Chats**: Оновлення списку чатів
- **Clear Messages**: Очищення вікна повідомлень

#### ⚙️ Settings
- **Font Size**: Зміна розміру шрифту повідомлень

#### ❓ Help
- **About**: Інформація про програму
- **How to get API credentials**: Інструкції по отриманню API даних

### Автоматично створювані файли
- `telegram_config.json` - Збережені налаштування
- `telegram_session.session` - Сесія Telegram (містить авторизаційні дані)

## 🔧 Конфігурація

Програма автоматично зберігає налаштування у файлі `telegram_config.json`:

```json
{
  "api_id": "your_api_id",
  "api_hash": "your_api_hash",
  "phone": "your_phone_number",
  "session_name": "telegram_session",
  "theme": "light",
  "font_size": 10,
  "window_geometry": "1200x800"
}
```

## 🔒 Безпека

⚠️ **ВАЖЛИВО**: 
- Зберігайте ваші API credentials в безпеці
- Не діліться ними з іншими людьми
- Файл `telegram_session.session` містить авторизаційні дані - не передавайте його
- Не публікуйте файл `telegram_config.json` у відкритих репозиторіях

## 🐛 Вирішення проблем

### Помилка підключення
```
Failed to connect: [Error details]
```
**Рішення**:
- Перевірте правильність API ID та API Hash
- Перевірте інтернет-з'єднання
- Переконайтесь що номер телефону введено у міжнародному форматі

### Код підтвердження
```
Please enter the code you received: 
```
**Рішення**:
- Перевірте SMS або повідомлення в Telegram
- Введіть 5-значний код
- Якщо код не приходить, спробуйте ще раз через хвилину

### Помилка двофакторної автентифікації
```
Two-step verification is enabled
```
**Рішення**:
- Введіть пароль від двофакторної автентифікації
- Якщо забули пароль, відновіть його через офіційний Telegram

### Помилка tkinter
```
tkinter не доступний
```
**Рішення**:
```bash
sudo apt install python3-tk
```

### Помилка telethon
```
No module named 'telethon'
```
**Рішення**:
```bash
pip install telethon --break-system-packages
```

## 🎨 Персоналізація

### Зміна теми
Поки що підтримується тільки світла тема, але можна легко додати темну:

```python
# У telegram_config.py додайте методи для теми
def get_theme_colors(self):
    if self.get_theme() == 'dark':
        return {
            'bg': '#2b2b2b',
            'fg': '#ffffff',
            'select_bg': '#404040'
        }
    return {
        'bg': '#ffffff',
        'fg': '#000000',
        'select_bg': '#e0e0e0'
    }
```

### Додавання нових функцій
Для додавання нових функцій:

1. Додайте метод у клас `TelegramClientGUI`
2. Створіть GUI елементи в `setup_gui()`
3. Додайте обробники подій
4. Використовуйте `asyncio.run_coroutine_threadsafe()` для асинхронних операцій
