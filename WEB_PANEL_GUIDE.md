# 🌐 Веб-панель розробника YouTube Bot

## 🎯 Огляд

Ви успішно створили потужну веб-панель розробника для управління YouTube Telegram ботом! Панель надає зручний інтерфейс для:

- 📊 **Моніторингу** статистики та активності бота
- ⚙️ **Налаштування** конфігурації без редагування файлів
- 👥 **Управління** користувачами та їх правами
- 📥 **Відстеження** завантажень та їх статусу
- 📋 **Перегляду** логів в реальному часі

## 🚀 Швидкий старт

### 1. Запуск панелі

```bash
# Активуйте віртуальне середовище
source venv/bin/activate

# Запустіть веб-панель
python start_web_panel.py
```

### 2. Доступ до панелі

- **URL:** http://localhost:5000
- **Логін:** admin
- **Пароль:** admin123

⚠️ **Важливо:** Змініть пароль за замовчуванням у продакшені!

## 📋 Функціонал панелі

### 🏠 Панель управління (Dashboard)
- **Статистика користувачів** - загальна кількість, активні, заблоковані
- **Статистика завантажень** - сьогодні, успішні, помилки
- **Статус бота** - онлайн/офлайн, час роботи
- **Швидкі дії** - тест YouTube API, перезапуск бота
- **Графіки активності** - розподіл завантажень по типах

### ⚙️ Конфігурація
**Основні налаштування:**
- Назва бота
- Telegram Bot Token (приховано)

**Параметри завантаження:**
- Максимальний розмір файлу (МБ)
- Максимальна тривалість відео (хвилини)
- Якість відео за замовчуванням (360p-1080p)
- Якість аудіо за замовчуванням (128-320 kbps)
- Шляхи для завантажень та тимчасових файлів

**Обмеження та захист:**
- Максимум завантажень на годину на користувача
- Максимум запитів на хвилину

**Логування:**
- Рівень логування (DEBUG, INFO, WARNING, ERROR)
- Файл для збереження логів

**Додаткові функції:**
- ✅ Тест конфігурації
- 📥 Експорт налаштувань в JSON
- 📋 Копіювання конфігурації
- 🔄 Оновлення налаштувань в реальному часі

### 👥 Управління користувачами
**Статистика:**
- Всього користувачів
- Активні сьогодні
- Заблоковані
- Нові за тиждень

**Список користувачів:**
- ID, ім'я, username, Telegram ID
- Кількість завантажень
- Остання активність
- Статус (активний/заблокований)

**Функції:**
- 🔍 Пошук за ім'ям або ID
- 🔽 Фільтрація за статусом
- 📊 Сортування за різними критеріями
- 👁️ Детальний перегляд користувача
- 🚫 Блокування/розблокування
- 📥 Експорт списку користувачів

### 📥 Моніторинг завантажень
**Статистика:**
- Всього завантажень
- Успішні завантаження
- Помилки
- Завантаження сьогодні

**Список завантажень:**
- ID, назва відео, користувач
- Тип (відео/аудіо), розмір, тривалість
- Статус (завершено/помилка/в процесі)
- Дата створення

**Фільтри:**
- 🔍 Пошук за назвою або користувачем
- 📱 Тип файлу (відео/аудіо)
- ✅ Статус завантаження
- 📅 Період (сьогодні/тиждень/місяць)

**Дії:**
- 👁️ Детальний перегляд завантаження
- 📥 Завантаження готового файлу
- 🗑️ Видалення завантаження

### 📋 Перегляд логів
**Налаштування перегляду:**
- 🎯 Фільтр за рівнем логування
- 🔍 Пошук в логах
- 📊 Кількість рядків (50-1000)
- 🔄 Авто-оновлення кожні 5 секунд

**Функції:**
- 📥 Експорт логів у файл
- 🧹 Очищення відображення
- ⬇️ Прокрутка до кінця/початку
- 👁️ Детальний перегляд лог-запису
- 📋 Копіювання лог-записів

**Особливості:**
- 🎨 Кольорове виділення за рівнем логування
- 🔍 Підсвічування результатів пошуку
- ⏰ Показ часу останнього оновлення
- 🔗 Індикатор стану з'єднання

## 🔧 API Endpoints

Панель надає REST API для інтеграції з іншими системами:

### Конфігурація
- `GET /api/config` - Отримати поточні налаштування
- `POST /api/config` - Оновити налаштування

### Користувачі
- `GET /api/users` - Статистика та список користувачів

### Завантаження
- `GET /api/downloads` - Статистика та список завантажень

### Логи
- `GET /api/logs?lines=100` - Отримати останні логи

### Утиліти
- `GET /api/test-youtube` - Тестувати YouTube сервіс
- `POST /api/restart-bot` - Перезапустити бота

## 🛠️ Налаштування для продакшену

### 1. Зміна пароля адміністратора

```python
# У файлі web_panel/app.py
from werkzeug.security import generate_password_hash

# Замініть цей рядок:
ADMIN_USERS = {
    'admin': generate_password_hash('ваш_новий_пароль')
}
```

### 2. Встановлення SECRET_KEY

```bash
# Створіть змінну середовища
export WEB_SECRET_KEY="ваш-складний-секретний-ключ"
```

### 3. Конфігурація nginx (опціонально)

```nginx
server {
    listen 80;
    server_name your-domain.com;
    
    location / {
        proxy_pass http://127.0.0.1:5000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
```

### 4. Systemd сервіс (опціонально)

```ini
[Unit]
Description=YouTube Bot Web Panel
After=network.target

[Service]
Type=simple
User=your-user
WorkingDirectory=/path/to/project
Environment=PATH=/path/to/project/venv/bin
ExecStart=/path/to/project/venv/bin/python start_web_panel.py
Restart=always

[Install]
WantedBy=multi-user.target
```
