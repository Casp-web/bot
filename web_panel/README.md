# 🌐 YouTube Bot - Панель розробника

Веб-інтерфейс для налаштування та моніторингу YouTube Telegram бота.

## ✨ Можливості

### 📊 Панель управління
- Статистика користувачів та завантажень
- Моніторинг стану бота в реальному часі
- Графіки та діаграми активності
- Швидкі дії для управління ботом

### ⚙️ Конфігурація
- **Основні налаштування бота** - назва, токен
- **Параметри завантаження** - максимальний розмір файлу, тривалість відео
- **Якість за замовчуванням** - відео (360p-1080p), аудіо (128-320 kbps)
- **Обмеження та захист** - ліміти на завантаження, захист від спаму
- **Логування** - рівень логування, файли логів
- **Експорт/імпорт** налаштувань у форматі JSON

### 👥 Управління користувачами
- Перегляд списку всіх користувачів
- Детальна інформація про кожного користувача
- Блокування/розблокування користувачів
- Статистика активності та завантажень
- Пошук та фільтрація користувачів
- Експорт даних користувачів

### 📥 Моніторинг завантажень
- Список всіх завантажень з деталями
- Фільтрація за типом (відео/аудіо), статусом, періодом
- Перегляд прогресу завантажень
- Управління файлами завантажень
- Статистика успішних та неуспішних завантажень

### 📋 Перегляд логів
- Перегляд логів бота в реальному часі
- Фільтрація за рівнем логування (DEBUG, INFO, WARNING, ERROR)
- Пошук в логах
- Авто-оновлення логів
- Експорт логів у файл
- Детальний перегляд кожного лог-запису

## 🚀 Запуск

### Автоматичний запуск
```bash
# З кореневої директорії проекту
python start_web_panel.py
```

### Ручний запуск
```bash
# Встановити залежності
pip install flask>=3.0.0 flask-login>=0.6.3

# Запустити веб-панель
cd web_panel
python app.py
```

## 🔐 Доступ

- **URL:** http://localhost:5000
- **Логін:** admin
- **Пароль:** admin123

⚠️ **Важливо:** Змініть пароль за замовчуванням у продакшені!

## 🔧 API Endpoints

### Конфігурація
- `GET /api/config` - Отримати поточну конфігурацію
- `POST /api/config` - Оновити конфігурацію

### Користувачі
- `GET /api/users` - Отримати статистику користувачів

### Завантаження
- `GET /api/downloads` - Отримати статистику завантажень

### Логи
- `GET /api/logs` - Отримати логи бота

### Утиліти
- `GET /api/test-youtube` - Тестувати YouTube сервіс
- `POST /api/restart-bot` - Перезапустити бота

## 🔒 Безпека

### Аутентифікація
- Базова аутентифікація з Flask-Login
- Сесійне управління
- Захист від CSRF (рекомендується додати)

### Рекомендації для продакшену
1. **Змініть пароль за замовчуванням**
2. **Використовуйте HTTPS**
3. **Обмежте доступ до панелі** (firewall, VPN)
4. **Встановіть складний SECRET_KEY**
5. **Використовуйте реверс-проксі** (nginx)
6. **Регулярно оновлюйте залежності**

## 🛠️ Налаштування

### Змінні середовища
```bash
# Секретний ключ для Flask
WEB_SECRET_KEY=your-secret-key-here

# Пароль адміністратора (хешований)
ADMIN_PASSWORD_HASH=your-hashed-password
```

### Конфігурація у файлі
Відредагуйте `app.py` для зміни:
- Порту (за замовчуванням 5000)
- Хоста (за замовчуванням 0.0.0.0)
- Режиму відладки

## 🐛 Відладка

### Логи Flask
Логи Flask виводяться в консоль. Для збереження в файл:
```python
import logging
logging.basicConfig(filename='web_panel.log', level=logging.INFO)
```

### Режим відладки
За замовчуванням увімкнений режим відладки (`debug=True`). 
Вимкніть у продакшені!
