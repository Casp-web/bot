# Telegram Tkinter Client - Налаштування

## Опис
Це простий Telegram клієнт, створений на tkinter з використанням бібліотеки Telethon. Клієнт дозволяє:
- Авторизуватися в Telegram
- Переглядати список чатів
- Читати та відправляти повідомлення
- Отримувати повідомлення в реальному часі

## Встановлення залежностей

```bash
pip install -r requirements.txt
```

## Отримання API ключів

1. Перейдіть на https://my.telegram.org
2. Увійдіть зі своїм номером телефону
3. Перейдіть до "API development tools"
4. Створіть новий додаток:
   - App title: Будь-яка назва (наприклад, "My Tkinter Client")
   - Short name: Коротка назва (наприклад, "tkclient")
   - Platform: Desktop
   - Description: Опис додатку

5. Скопіюйте `api_id` та `api_hash`

## Запуск клієнта

```bash
python telegram_client.py
```

## Використання

1. **Авторизація:**
   - Введіть API ID, API Hash та номер телефону
   - Натисніть "Login"
   - Введіть код підтвердження, який прийде в Telegram

2. **Робота з чатами:**
   - Після авторизації завантажиться список чатів
   - Виберіть чат зі списку для перегляду повідомлень

3. **Відправка повідомлень:**
   - Виберіть чат
   - Введіть повідомлення в поле вводу
   - Натисніть "Send" або Enter

## Функції

- **Автоматичне збереження налаштувань** - дані для входу зберігаються в `telegram_config.json`
- **Отримання повідомлень в реальному часі** - нові повідомлення автоматично з'являються в чаті
- **Простий інтерфейс** - зручний tkinter інтерфейс
- **Безпека** - API Hash приховується при введенні

## Структура файлів

- `telegram_client.py` - основний файл клієнта
- `telegram_config.json` - файл з налаштуваннями (створюється автоматично)
- `session_name.session` - файл сесії Telegram (створюється автоматично)

## Безпека

⚠️ **Важливо:**
- Не передавайте API ключі третім особам
- Файл `telegram_config.json` містить конфіденційні дані
- Файл `session_name.session` містить дані авторизації

## Відомі обмеження

- Підтримуються тільки текстові повідомлення
- Немає підтримки медіафайлів
- Немає підтримки групових чатів з великою кількістю учасників
- Інтерфейс може бути повільним при великій кількості чатів

## Розширення функціоналу

Можна додати:
- Підтримку медіафайлів (фото, відео, документи)
- Пошук по повідомленнях
- Налаштування теми інтерфейсу
- Підтримку стикерів та емодзі
- Експорт повідомлень
- Налаштування сповіщень

## Технічна інформація

- **Бібліотека:** Telethon (офіційна Python бібліотека для Telegram API)
- **GUI:** Tkinter (стандартна Python бібліотека)
- **Асинхронність:** asyncio + threading для роботи з UI
- **Збереження даних:** JSON для налаштувань, SQLite для сесії
